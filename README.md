## 👤 Sobre Mim

Tenho **16 anos**, estou no 2º ano do ensino médio e já construo aplicações reais que rodam em produção.

Não aprendo só lendo documentação — aprendo **construindo**. Cada projeto é um problema pra resolver, um sistema pra arquitetar, uma experiência de usuário pra refinar. Sou apaixonado por tecnologia desde criança, e essa curiosidade virou obsessão por entender como os sistemas funcionam por dentro.

Fluente em inglês. Estudando JavaScript no SENAI. Focado em entrar no mercado tech e, eventualmente, construir meu próprio SaaS.
