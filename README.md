### 🔥 [`bonds-agence`](https://github.com/gomeslxy/bonds-agence) — E-commerce Full-Stack em Produção

> *"Não é apenas uma loja — é um movimento."*

Uma plataforma de e-commerce de streetwear construída do zero, com stack moderno e foco em **performance, segurança e experiência do usuário**. Rodando em produção em [bonds-agence.vercel.app](https://bonds-agence.vercel.app).

```
Tech Stack: Next.js 14 · TypeScript · Supabase · PostgreSQL
            Upstash Redis · Stripe · Framer Motion · Vercel
```

| Feature | Descrição |
|---|---|
| 🛒 **Checkout Inteligente** | Fluxo de compra com validação de CPF e máscaras de input dinâmicas |
| 🔐 **Auth SSR** | Autenticação segura via Next.js Server Actions, sem expor tokens no client |
| 🛡️ **Rate Limiting** | Anti-brute force via Upstash Redis em todas as rotas críticas |
| 📊 **Admin Dashboard** | Gestão de estoque, pedidos em tempo real e criação de cupons |
| 📡 **Realtime Engine** | Sincronização instantânea entre banco e UI |
| 📧 **Email Automation** | Notificações de pedido via Supabase Edge Functions |
| 🔒 **RLS no PostgreSQL** | Row Level Security — usuários só acessam seus próprios dados |
